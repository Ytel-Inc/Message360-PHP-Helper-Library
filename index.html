Welcome to the Message360 PHP Helper Library

This is home to the Official PHP Message360 REST API and InboundXML Libraries.

Example:

<?php
## First we must import the actual Message360 library
require_once '../../library/message360.php';

# Now what we need to do is instantiate the library
$Message360 = Message360::getInstance();

# Message360 REST API credentials are required
$Message360 -> setOptions(array( 
    'account_sid'       => 'xxxx-xxxxxx-xxxxxx', 
    'auth_token'        => 'xxxx-xxxxxx-xxxxxx',
    'response_to_array' =>true,
    ));
?>
